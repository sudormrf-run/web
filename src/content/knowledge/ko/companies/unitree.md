## Overview

Unitree Robotics는 중국의 로봇 하드웨어 제조사로, 저렴하면서도 고성능인 사족보행 로봇과 휴머노이드를 생산합니다. $16,000 휴머노이드 G1으로 가격 혁신을 이끌고 있습니다.

| 항목 | 내용 |
|------|------|
| 본사 | 항저우, 중국 |
| 설립 | 2016년 |
| CEO | Wang Xingxing |
| 기업가치 | ~$7B (보도 기반 추정) |
| 핵심 제품 | Go2, B2, G1, H1 |

## G1 휴머노이드

### 스펙

| 항목 | 수치 |
|------|------|
| 높이 | 127cm (8세 아동 크기) |
| 무게 | 35kg |
| 자유도 | 23-43 DoF (구성에 따라) |
| 보행속도 | 2m/s (4.5 MPH) |
| 센서 | 3D LiDAR, Intel RealSense D435 |
| 손 | 힘 제어 가능한 손가락 |

### 특징

- **접이식 디자인**: 컴팩트하게 운반 가능
- **저렴한 가격**: $16,000 (기존 휴머노이드 대비 혁신적)
- **강화학습 기반**: 이미테이션/강화 학습 적용

| 항목 | 수치 |

- 복잡한 지형 자율 보행
