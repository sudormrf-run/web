## Overview

Unitree Robotics is a Chinese robot hardware manufacturer producing affordable yet high-performance quadruped robots and humanoids. They are leading price innovation with the $16,000 G1 humanoid.

| Item | Details |
|------|---------|
| Headquarters | Hangzhou, China |
| Founded | 2016 |
| CEO | Wang Xingxing |
| Valuation | ~$7B (estimated from reports) |
| Core Products | Go2, B2, G1, H1 |

## G1 Humanoid

### Specs

| Item | Value |
|------|-------|
| Height | 127cm (size of an 8-year-old) |
| Weight | 35kg |
| DoF | 23-43 DoF (depending on configuration) |
| Walking Speed | 2m/s (4.5 MPH) |
| Sensors | 3D LiDAR, Intel RealSense D435 |
| Hands | Force-controllable fingers |

### Features

- **Foldable Design**: Compact for transportation
- **Affordable Price**: $16,000 (revolutionary compared to other humanoids)
- **RL-Based**: Imitation/reinforcement learning applied
